# 🚀 Job Portal Web Application

A full-stack **Job Portal platform** built using **React (Vite) frontend** and **Django REST Framework backend**, designed to connect job seekers with recruiters through a fast, scalable, and modern web application.

---

## 📌 Project Overview

This project allows users to:
- Register and securely log in
- Browse available job listings
- Apply for jobs instantly
- Track application status
- Provides backend structure for recruiters to manage job postings

The system follows a **REST API architecture** with a clean separation between frontend and backend.

---

## ✨ Features

### 👨‍💼 Job Seekers
- User authentication (Login / Register)
- View all available jobs
- Apply for jobs easily
- Track application status
- Responsive UI for all devices

### 🏢 Recruiter Side (Backend Ready)
- Create and manage job posts
- View applicants per job
- Manage recruitment workflow

### ⚙️ System Features
- REST API integration (Django REST Framework)
- Secure authentication system
- Environment variable configuration (.env support)
- CORS enabled for frontend communication
- Production-ready deployment setup

## 🛠️ Tech Stack

### 💻 Frontend
- React 19
- Vite
- React Router DOM 7
- Tailwind CSS 4

### 🔧 Backend
- Django 6
- Django REST Framework
- PostgreSQL
- Gunicorn
- django-cors-headers
- Whitenoise

---

## 🗄️ Database
- PostgreSQL (Hosted on Render)
- Stores users, jobs, and applications data

---

## 🔐 Security
- Environment variables for sensitive data
- Secure SECRET_KEY handling
- CORS protection enabled
- Production-safe Django configuration

---

## 🌐 Deployment

- **Frontend:** Vite build (static hosting ready)
